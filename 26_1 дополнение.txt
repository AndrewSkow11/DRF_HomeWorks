На основании трех эндпоинтов ты можешь создать три сервисные функции:
		• для создания продукта
		• для создания цены
		• для создания сессии и получения ссылки на оплату

При создании курса тебе нужно создавать продукт в stripe (используя первую функцию) и записывать id продукта от stripe в модель курса

Остальные две функции ты должен использовать при создании платежа - берем продукт, создаем цену и сессию - отдаем пользователю ссылку на оплату
